# Cisco-Packet-Tracer-Labs
Cisco Packet Tracer projects demonstrating enterprise network design, routing, switching, VLAN configuration, and network security concept

This repository contains networking labs created using Cisco Packet Tracer.

## Topics Covered
- VLAN
- Inter-VLAN Routing
- Static Routing
- OSPF
- ACL
- DHCP
- Network Troubleshooting

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI

These labs were created for CCNA practice and hands-on networking experience.
